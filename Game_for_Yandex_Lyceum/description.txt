Данный проект является школьным и давно сданным куда надо, так что его можно считать школьным-незавершенным.
Цель проекта - создать игру рогалик.
Реализованные механики - Генерация 20 различных комнат(заранее созданных), возможность изменения характеристек персонажа во время игры, стрельба по монстрам (но для стрельбы нужны патроны-камни).
Для перехода в следующую комнату нужно нажать на все кнопки, если их нет то через маленький промежуток времени двери откроются.
Управление: Передвижение на WASD, на пробел стрелять, esc включает паузу, Q - использовать витаминки.
Для запуска игры нужно запустить файл run.py
